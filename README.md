# RdpGuard

## Overview

RdpGuard is a security application designed to protect Windows systems from unauthorized Remote Desktop Protocol (RDP) access attempts. It monitors login activity and blocks IP addresses that exhibit suspicious behavior, helping to reduce the risk of brute force attacks and unauthorized access.

## Key Features

**Monitors RDP login attempts in real time**  
**Automatically blocks IP addresses after multiple failed authentication attempts**  
**Supports integration with Windows Firewall for automatic rule management**  
**Provides detailed logging and reports on blocked IPs and attack sources**  
**Configurable thresholds for failed login detection**  
**Lightweight operation with minimal system resource usage**  
**Supports IPv4 and IPv6 addresses**

## Why RdpGuard?

RdpGuard offers a clear and straightforward approach to securing RDP services on Windows devices. Its focus on reliable detection and blocking mechanisms ensures that unauthorized access attempts are efficiently managed. The software is designed to be user-friendly, with transparent configuration options and detailed monitoring to support system administrators in maintaining secure remote access environments.

## Benefits

Improves overall system security by reducing exposure to unauthorized remote login attempts  
Helps maintain network integrity through automated IP blocking  
Provides clear insights into RDP access patterns and security events  
Operates quietly in the background without interrupting legitimate users  
Allows customization to fit various security policies and organizational requirements

## Compatibility

This repository is developed specifically for Windows operating systems. It delivers stable performance and efficient operation within the Windows environment to support ongoing protection of RDP services.
